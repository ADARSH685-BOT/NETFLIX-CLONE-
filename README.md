# Netflix Clone 

This is a **Netflix landing page clone** created using **HTML and CSS only**.  
The project focuses on building a clean and modern UI similar to Netflix to practice
frontend layout, styling, and responsive design concepts.

 This is a **static UI project**. Buttons and Sign In functionality are not active
because JavaScript and backend are not used.

---

## Features

- Netflix-style homepage UI
- Header with Netflix logo and Sign In button
- Hero banner section
- Movie poster sections
- Clean and simple design
- Beginner-friendly project

---

## Tech Stack

- **HTML5**
- **CSS3**
  - Flexbox
  - Basic responsiveness
  - Custom styling
